# 项目简介

## 核心功能
- 基于YOLOv8+PaddleOCR的车牌检测系统
- 支持图片、视频及摄像头进行检测
- 支持选择不同检测模型
- 界面实时显示目标位置、目标总数、置信度等信息
- 支持批量检测并在界面直接查看所有检测结果
- 支持检测结果保存
- 可自定义置信度和IOU阈值
- 可选择是否显示标签和原图

## 适用场景
- 车牌识别应用开发
- 车辆管理系统
- 智能监控系统
- 交通流量分析
- 停车场管理系统
- 安防监控系统

## 技术特点
- 使用Pycharm+Anaconda或VSCode+Anaconda开发环境
- 基于Python 3.9，依赖opencv-python、PyQt5、ultralytics、torch1.9等技术
- 包含433张车牌区域识别数据集，标签为'chepai'共1类
- 预训练模型可直接使用，配置环境后即可运行
- 提供完整的程序文件、UI界面源文件、图标、数据集图片和运行教程
